# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Vehicle Dashboard System
- Complete sensor integration (GPS, temperature, voltage)
- Web interface with real-time control
- Multiple display modes with animations
- Statistics tracking for all sensors
- RTC time synchronization
- Touch button control
- Configurable settings via web interface

### Features
- Real-time GPS speed monitoring
- Battery voltage tracking
- Temperature sensing
- OLED display with animated eyes
- WiFi access point mode
- Responsive web dashboard
- Buzzer alert system
- Data statistics (max/min/avg)
